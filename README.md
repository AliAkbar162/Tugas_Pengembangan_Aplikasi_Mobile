# Tugas_Pengembangan_Aplikasi_Mobile_Minggu 2
# cara menjalankan aplikasi
# Pertama
download folder di github dan ekstrak file tersebut

# kedua
buka aplikasi android studio, pada bagian file pilih open dan arahkan ke folder project yang telah diekstrak tadi kemudian klik ok dan tunggu untuk gradle sync-nya selesai.

# ketiga 
setelah file terbuka, pastikan bahwa gradle telah benar benar sudah tersinkronisasi dengan cara klik file kemudian klik sync project with gradle files lalu tunggu hingga prosesnya selesai.

# keempat
pastikan device yang akan digunakan telah siap baik itu virtual device ataupun device langsung. jika sudah pilih configurasi composeApp di bagian atas lalu klik tombol RUN yang ada disampingnya kemudian tunggu proses build aplikasinya selesai.

# kelima
Setelah selesai, aplikasi akan otomatis terbuka di device yang sudah disiapkan. didalam aplikasi, terdapat tombol mulai stream untuk memulai berita yang akan muncul, stop untuk menghentikan aplikasi. tersedia juga filter bereita yang ada dalam bentuk kategori yang ditampilkan (semua/teknologi/olahraga/dll). berita juga dapat dibaca dengan menekan salah satu berita yang ingin dibaca, yang dimana jika ditekan akan mengarahkan ke tempat berita lengkapnya. jika ingin keluar dan membaca berita yang lain bisa menekan tombol kembali yang ada.
